## 담당한 업무
- [기업회원] 지원자 탭 
  - 기업에 지원한 전체 지원자 목록 확인 가능  
  - 채용공고별 지원한 지원자 목록 확인 가능 - 채용공고 메뉴는 동적으로 생성  
  - 지원자의 이름, 직무로 검색 가능, 페이징 처리
  - 이력서를 열람한 지원자 목록 확인 가능 - 지원자 이력서 확인 가능  
  
- [기업회원] 매치업 탭
  - 야너두 모든 회원의 프로필을 확인 가능
  - 직무별 검색, 찜하기, 페이징 기능
  - 이력서 미리보기로 회원의 이력서를 확인 가능 (마스킹 처리된 이력서)
  - 열람권을 사용하여 이력서 상세보기 가능  
  
- [기업회원] 모집공고 탭
  - 모집공고 등록 
  - 모집공고 조회 가능
  - 모집공고 수정 가능
  - 모집공고 삭제 가능
